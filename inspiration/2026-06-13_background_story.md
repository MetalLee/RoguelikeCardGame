# 对游戏背景设定的补充以及卡牌、战斗机制的重新设计

## 游戏设定

### 游戏名
《剑与黑塔》

### 世界观
一百年前，随着人类最后一颗原子弹的爆发，人类文明迎来了首次自我毁灭，世界化为一片荒芜。

「黑色的的山」、「白色的天」、「灰色的光」，世界在一夜之间、不知为何地失去了「色彩」，人类相信是神的惩罚。

人类失去色彩的那一夜，世界各地出现一座座神秘的「黑塔」。有人认为里头拥有巨额宝藏、有人认为那是外星人侵略地球的据点，里面有人类未知的科技文明。

而有一批人类却坚决相信：「黑塔」，是夺回世界色彩的关键。

黑塔出现后，除了黑塔周围的地区荒漠化日益严重，而越靠近黑塔的地区，则越有生机。而且围绕着黑塔的周边的区域发生了许多光怪陆离的事与超自然现象。

为了探索造成世界怪异的源头，和夺回人类世界的色彩。一批志同道合的人成立了无政府组织「勇者联盟」，目的是攻克黑塔。但是随着时间的推移，联盟也日益腐败，出现了大批取得职业勇者证的人玩弄特权的现象…

而人类在重建文明的过程中，「色彩」逐渐成为了传说之物，国家概念则变得非常孱弱,新生的人类社会崇尚弱肉强食。资源争夺、领土问题、种族斗争、社会运动…旧社会的一幕幕仍在不断重演…

### 故事简介
「左」曾经是知名勇者「茵」的剑——左轮剑所诞生的剑灵，二人曾因理念问题分道扬镳。后来在某次意外当中，「左」作为剑灵面临着消失的危机，为了救下「左」，「茵」通过某种禁忌魔术牺牲了自己，把自己的肉身和「左」的灵结合，令「左」成为了人类。成为人类的「左」感到内疚与后悔，决定一个人穷尽一生去实现「茵」生前的愿望——将世界恢复色彩。「左」带着「茵」遗留下的、自己曾经的容器「左轮剑」进行了孤单的旅行。而在前往一座座黑塔解开人类失去色彩之谜的过程中，遇到了各式各样的人与事的「左」，亦重新体会到「茵」当年为了践行勇者职责而不被理解的心情。而「左」却不知，「茵」的灵仍在「左轮剑」中沉睡…

### 角色设定
单主角，擅长使用各种冷武器与高科技武器。
默认主手武器：左轮剑，剑与枪的结合，通过剑术斩击可以将「彩能」充入剑内，然后转变为枪形态以「彩能」为子弹倾泻伤害。
默认副手武器：机械臂，高科技义肢，能生成粒子护盾同时生产「彩能」，并可对敌人形成强力的控制效果。

除了默认的左轮剑与机械臂，在游戏过程中可以解锁更多的武器

初始行动点为3，能存储的「彩能」上限为6（配合左轮设定），「彩能」跨回合不保留。

### 卡牌机制

#### 卡牌分类
行动牌：「彩能」的主要生成手段，可被「色彩」附魔产生特殊效果

终结牌：消耗「彩能」造成强大的伤害或效果，并根据所有消耗的「彩能」附加的「色彩」产生特殊效果

#### 初始卡牌
每种武器有8张初始卡牌，每轮游戏开始时，主角需要从主手武器选6张+副手武器选4张，组成10卡牌的初始卡组。

左轮剑：
- 斩击*4（行动牌）—— 消耗1行动点，对单体敌人造成?点伤害，获得1点彩能
- 二连斩（行动牌）—— 消耗1行动点，对单体敌人造成?*2点伤害，获得2点彩能
- 快速突刺（行动牌）—— 消耗0行动点，对单体敌人造成?点伤害，获得1点彩能
- 重炮（终结牌）—— 消耗3点彩能，对单体敌人造成?点伤害
- 子弹风暴（终结牌）—— 消耗X点彩能，对随机敌人造成X次?点伤害（X为当前彩能数）

机械臂：
- 格挡*4（行动牌）—— 消耗1行动点，获得?点护盾，获得1点彩能
- 束缚（行动牌）—— 消耗1行动点，使单体目标直到下回合无法行动，获得1点彩能
- 重压（行动牌）—— 消耗1点行动点，使单体目标直到下回合受到伤害增加50%
- 横扫千军（终结牌）—— 消耗4点彩能，对所有敌人造成?点伤害
- 反击（终结牌）—— 消耗2点彩能，对单体敌人造成等于当前护甲值的伤害

#### 稀有度
普通、罕见、稀有，影响每场战斗后的三选一中的出现概率

### 颜色机制
初版设置5种「色彩」，「色彩」可以附魔到「行动牌」，使「行动牌」产生的「彩能」带上「色彩」，「终结牌」消耗「彩能」时，会将所有「彩能」的「色彩」效果叠加，产生更强大的特殊效果

- 红色：力量，附加伤害效果

- 黄色：速度，增加卡牌释放次数

- 蓝色：防卫，附加护盾效果

- 绿色：生命，附加回复效果

- 紫色：神秘，附加翻倍效果

### 遗物设计：

通过事件或击败精英怪、boss可以获得遗物，遗物描述中隐晦表达剧情

遗物可以为主角添加被动玩法，精英或事件掉落的普通遗物以增强能力为主，boss掉落的更强力遗物可以改变玩法

### 奖励机制

每场战斗后：
- 随机获得一个「色彩」碎片，可以选择为卡组中一张未附魔的「行动卡」附魔
- 可以三选一为主武器或副武器添加一张卡牌

精英战：
- 额外奖励一个普通遗物

Boss战（每关结束）：   
- 额外奖励一个Boss遗物，掉落特有Boss卡牌
- 获得随机颜色的「色彩」核心，提升「色彩」对卡牌的附魔效果

### 事件机制

通过在地图上的问号事件，结合二选一的奖励机制，来进行碎片化叙事，塑造世界观

## 2026-06-13 与 Codex 的设计确认 Q&A

本节记录本次灵感整理后，开发者对“如何修改现有 design/ 设计文档”的正式确认。后续设计文档应以这些回答为准。

### Q1：游戏名与世界观是否正式落地？

A：游戏名与世界观按《剑与黑塔》、核后失色世界、黑塔、勇者联盟、左与茵设定正式落地。

### Q2：叙事优先级要不要上调？

A：把角色旅途与黑塔谜团提升为重要卖点，但不压过卡牌战斗。剧情主要通过事件、卡牌、遗物的文字描述隐晦表达，玩家通过它们了解剧情全貌，体现碎片化叙事。

### Q3：彩能与旧连锁层数的关系是什么？

A：彩能只是连锁层数的世界观命名，规则大体保留；但是不再保留阈值 3/5/8。彩能可以被附魔。

### Q4：3/5/8 固定阈值是否保留？

A：取消 3/5/8 固定阈值，终结牌消耗彩能按卡牌具体标注。

### Q5：卡牌类型是否从三类改成两类？

A：删除“技能牌”，全部归入行动牌与终结牌。

### Q6：初始牌组规则是否改为武器选牌？

A：立即把设计改为武器选牌结构，MVP 也改，但先只支持左轮剑 + 机械臂。

### Q7：奖励系统是否加入“色彩碎片 / 附魔”？

A：新奖励机制替代当前卡牌包，MVP 也要改动。

### Q8：五色机制是否作为新的核心构筑系统？

A：五色机制是核心机制，必须进入基础战斗设计，MVP 也要改动。

### Q9：武器系统是否视为装备系统？

A：武器是角色机制 / 卡池入口，不是装备掉落。每种装备拥有独立的战斗方式和卡池，并非传统意义上的装备系统。

### Q10：MVP 当前 6 场战斗是否需要改叙事包装？

A：是，连 MVP 敌人设定也要统一到黑塔世界观。

### Q11：Boss 奖励规则是否覆盖 MVP？

A：MVP 也加入简化 Boss 奖励。

### Q12：事件机制是否提前进入正式设计范围？

A：MVP 先打磨战斗系统，暂不做事件；但后续下一阶段优先做事件，而不是商店 / 路线图。

## 任务目标

将以下“通用色彩附魔机制 + MVP 左轮剑/机械臂 20 张独立卡牌设计 + 3 种 MVP 可行构筑流派”正式补充进 `design/` 目录。

当前设计基线为《剑与黑塔》：

- 起始主角：「左」。
    
- 起始武器：主手「左轮剑」、副手「机械臂」。
    
- 卡牌类型只保留：行动牌、终结牌。
    
- 不再设置独立“技能牌”类型。
    
- 行动牌主要生成彩能。
    
- 终结牌主要消耗彩能。
    
- 彩能基础上限为 6，跨回合不保留。
    
- 五色为：红、黄、蓝、绿、紫。
    
- 普通战斗奖励：随机色彩碎片 + 武器卡牌三选一。
    
- 第一版 MVP 需要支持至少 2 种可辨认的彩能 / 色彩构筑路线。
    

本次变更要强化一个核心设计原则：

> 色彩不是职业标签，而是卡牌用途的改写器。所有色彩都可以增强所有类型的卡牌，即使卡牌基础效果与颜色传统含义不同。

例如：

- 攻击终结牌「重炮」消耗蓝色彩能时，可以根据最终伤害获得护甲。
    
- 攻击终结牌「重炮」消耗绿色彩能时，可以根据最终伤害回复生命。
    
- 防御行动牌「格挡」被红色碎片附魔后，可以在获得防御的同时造成伤害。
    
- 控制牌被红色增强后，可以在施加控制的同时造成伤害。
    
- 防御终结牌被红色彩能增强后，可以把防御效果转化为群体伤害。
    
- 群攻牌被绿色彩能增强后，可以形成群体吸血感。

### 1. `design/01_core_gameplay/03_card_system.md`

重点补充：

#### 通用色彩附魔原则

新增或扩展“色彩附魔”章节，明确：

- 战斗后获得的色彩碎片用于附魔 1 张未附魔行动牌。
    
- 被附魔行动牌打出时，生成带有对应颜色的彩能。
    
- 行动牌自身也可以被颜色增强，不只是生成彩能。
    
- 终结牌消耗彩能时，根据消耗的彩能颜色逐一触发颜色增强。
    
- 同一颜色出现多次时，可以叠加增强，但 MVP 需要有叠加上限或明确数值边界。
    
- 所有颜色都可以作用于所有牌型，不要求“攻击牌只能吃红色、防御牌只能吃蓝色”。
    

#### 五色通用增强规则

补充一张表，内容大致如下：

| 色彩  | 通用含义 | 作用在攻击牌上         | 作用在防御牌上      | 作用在控制牌上    | 作用在终结牌上               |
| --- | ---- | --------------- | ------------ | ---------- | --------------------- |
| 红色  | 进攻转化 | 提高伤害或追加伤害       | 根据本次获得防御造成伤害 | 控制同时造成伤害   | 提高最终伤害，或将非伤害效果部分转化为伤害 |
| 黄色  | 节奏转化 | 增加卡牌释放次数        | 增加卡牌释放次数     | 增加卡牌释放次数   | 增加卡牌释放次数              |
| 蓝色  | 防护转化 | 根据造成伤害获得护甲 / 防御 | 提高获得的护甲值     | 控制同时获得护甲   | 根据最终伤害或最终效果值获得护甲 / 防御 |
| 绿色  | 生命转化 | 根据造成伤害回复生命      | 根据获得的护甲值回复生命 | 控制同时回复生命   | 根据最终伤害或最终效果值回复生命      |
| 紫色  | 效果翻倍 | 伤害翻倍            | 防御翻倍         | 异常效果或回合数翻倍 | 最终伤害或最终效果值翻倍          |

补充设计说明：

- 红色不是攻击专属，蓝色不是防御专属，绿色不是治疗专属。
    
- 玩家构筑的关键不是“拿什么颜色对应什么卡”，而是“用什么颜色改写这张卡的用途”。
    
- MVP 中应优先保证颜色预览清晰，而不是一次性做复杂联动。
    
- 黄色追加释放、紫色复制 / 翻倍属于高风险机制，但暂时不需要限制触发次数。
    
- 绿色回复应受最大生命限制，不允许超过生命上限。
    
- 蓝色护甲 / 防御应与现有防御清空规则保持一致，若需要跨回合保留必须单独写明。

3. 重压
    
    - 类型：行动牌
        
    - 建议费用：1 行动点
        
    - 彩能：建议获得 1 彩能

    - 用途：易伤、左轮爆发前置
        
4. 横扫千军
    
    - 类型：终结牌

新增牌：

7. 震荡掌

8. 牵引抓取
    
    - 类型：行动牌
        
    - 建议费用：1 行动点
        
    - 彩能：获得 1 彩能
        
    - 效果：对单体造成低伤害并施加“牵引”：本回合下一次单体攻击优先命中该目标且伤害提高

### 2. `design/01_core_gameplay/04_resource_economy.md`

补充“色彩作为资源转译器”的说明：

- 彩能不只是数量资源，也携带颜色信息。
    
- 色彩的价值体现在“改变卡牌结算结果”，而不是只给固定类型牌加成。
    
- 行动牌生成的彩能颜色来自该行动牌附魔。
    
- 终结牌消耗彩能时，按消耗彩能的颜色构成结算追加效果。
    
- 颜色结算需要支持预览：玩家在打出终结牌前能看到消耗哪些颜色、基础效果、追加效果和最终预估收益。
    
- 同色叠加需要定义 MVP 边界，例如：
    
    - 红色：每点红色提高一定伤害或追加一定伤害。
        
    - 蓝色：根据最终伤害 / 效果值获得一定比例护甲。
        
    - 绿色：根据最终伤害 / 效果值回复一定比例生命，但不能超过最大生命。
        
    - 黄色：增加卡牌的释放次数。
        
    - 紫色：复制、翻倍、异常延长，紫色获取渠道要更少，避免数值膨胀。
        

### 3. `design/02_content_systems/01_characters_and_archetypes.md`

新增或扩展 MVP 构筑原型，至少写入 3 种可行构筑。

#### 构筑一：蓝绿重炮流

核心定位：

> 用左轮剑打伤害，但不追纯红色，而是用蓝色和绿色把高伤害终结牌改造成护甲与治疗来源。

核心体验：

- 重炮不只是爆发牌，而是攻防治疗三合一。
    
- 蓝色重炮：造成伤害后，根据最终伤害获得护甲。
    
- 绿色重炮：造成伤害后，根据最终伤害回复生命。
    
- 红蓝绿重炮：高伤害 + 高护甲 + 治疗。
    

关键牌：

|武器|关键牌|作用|
|---|---|---|
|左轮剑|重炮|单体爆发核心|
|左轮剑|裂隙拔枪|击杀收割|
|左轮剑|装填斩|提高下一张终结牌伤害|
|左轮剑|枪剑校准|找终结牌，降低彩能压力|
|左轮剑|二连斩|快速生成彩能|
|左轮剑|快速突刺|0 费启动|
|机械臂|牵引抓取|锁定目标|
|机械臂|重压|易伤，提高最终收益|

推荐附魔：

|颜色|推荐对象|目的|
|---|---|---|
|蓝色|二连斩、装填斩、快速突刺|生成蓝色彩能，让重炮转化护甲|
|绿色|快速突刺、枪剑校准|生成绿色彩能，让重炮转化治疗|
|红色|装填斩、二连斩|少量提高伤害|
|紫色|重压、牵引抓取|放大终结前置收益|

典型回合：

1. 牵引抓取锁定目标。
    
2. 重压施加易伤。
    
3. 蓝色二连斩生成蓝色彩能。
    
4. 绿色快速突刺生成绿色彩能。
    
5. 装填斩强化下一张终结牌。
    
6. 重炮消耗蓝 + 绿 + 其他彩能，造成高伤害，并获得护甲与回复。
    

风险：

- 前期没有蓝绿彩能时只是普通重炮。
    
- 如果没有牵引抓取，单体爆发可能打错目标。
    
- 绿色回复需要生命上限限制。
    

#### 构筑二：红色机械防反流

核心定位：

> 用机械臂防御牌生成红色彩能，把防御牌改造成攻击牌，再用反击 / 护盾过载收尾。

核心体验：

- 红色格挡：获得防御，同时造成基于防御值的伤害。
    
- 红色能量护幕：获得大量防御，同时造成大量红色转化伤害。
    
- 反击消耗红 / 蓝彩能后，既能打伤害，又能再获得护甲。
    

关键牌：

|武器|关键牌|作用|
|---|---|---|
|机械臂|格挡|基础防御 + 红色伤害载体|
|机械臂|能量护幕|大防御 + 大红色转伤|
|机械臂|震荡掌|群体减攻|
|机械臂|反击|当前防御转单体伤害|
|机械臂|护盾过载|防御和彩能转群体伤害|
|机械臂|重压|易伤|
|左轮剑|快速突刺|0 费补彩能|
|左轮剑|装填斩|给终结补伤害|

推荐附魔：

|颜色|推荐对象|目的|
|---|---|---|
|红色|格挡、能量护幕|防御牌变输出牌|
|蓝色|格挡、震荡掌|保持防御稳定性|
|黄色|快速突刺、震荡掌|返还行动点 / 抽牌|
|紫色|重压、束缚|放大异常或延长控制|
|绿色|维修协议|低血线容错|

典型回合：

1. 红色格挡，获得防御并造成转化伤害。
    
2. 红色能量护幕，获得大量防御并造成转化伤害。
    
3. 震荡掌降低敌人攻击。
    
4. 重压施加易伤。
    
5. 反击消耗彩能，根据当前防御造成伤害。
    
6. 若反击消耗蓝色彩能，则根据最终伤害再次获得护甲。
    

风险：

- 成型前伤害较慢。
    
- 依赖红色防御牌。
    
- 敌人若有穿透 / 持续伤害，需要绿色续航补救。
    
- 红色防御牌的转伤比例建议先以 50%–100% 作为待调参范围。
    

#### 构筑三：黄紫弹幕调色流

核心定位：

> 利用黄色的行动点 / 抽牌 / 追加段数，以及紫色的放大 / 异常，让多段牌和 X 消耗终结牌反复触发色彩收益。

核心体验：

- 不是追求最高单次伤害，而是追求更多次出牌、更多段伤害、更多次色彩触发。
    
- 最适合围绕子弹风暴构筑。
    

关键牌：

|武器|关键牌|作用|
|---|---|---|
|左轮剑|子弹风暴|X 彩能弹幕终结|
|左轮剑|二连斩|多段伤害|
|左轮剑|快速突刺|0 费启动|
|左轮剑|回旋剑舞|群体多段、快速攒彩能|
|左轮剑|枪剑校准|找终结牌|
|左轮剑|贯穿弹线|利用控制状态追加伤害|
|机械臂|束缚|控制敌人|
|机械臂|牵引抓取|锁定关键目标|
|机械臂|重压|易伤，放大多段伤害|

推荐附魔：

|颜色|推荐对象|目的|
|---|---|---|
|黄色|快速突刺、二连斩、回旋剑舞|追加段数、抽牌、返还行动点|
|紫色|束缚、重压、贯穿弹线|放大异常，制造爆发窗口|
|红色|二连斩、回旋剑舞|提高多段总伤害|
|蓝色|任意高频产彩能牌|让弹幕终结后获得护甲|
|绿色|快速突刺或二连斩|让弹幕终结附带回血|

典型回合：

1. 黄色快速突刺，0 费启动，生成黄色彩能。
    
2. 黄色二连斩，造成两段伤害，生成黄色彩能。
    
3. 紫色重压，施加强化易伤，生成紫色彩能。
    
4. 牵引抓取锁定关键目标。
    
5. 枪剑校准寻找子弹风暴。
    
6. 子弹风暴消耗黄 + 黄 + 紫 + 红 + 蓝等彩能。
    
7. 黄色追加弹数 / 抽牌 / 返费；紫色施加裂痕或放大后续弹幕；蓝色根据最终总伤害获得护甲。
    

风险：

- 动画可能过长。
    
- 触发太多会让玩家看不懂。
    
- MVP 建议限制黄色每张牌最多触发 1 次返费 / 抽牌，子弹风暴最多追加 2 次额外弹幕。
    
- 紫色复制 / 放大不能造成无限循环。

### 5. `design/04_balance_data/00_balance_principles.md`

补充平衡原则：

- 防御牌吃红色转伤时，应先以“本次获得防御的一定比例”计算，不直接用总防御，避免滚雪球过快。
    
- 多段牌与红 / 黄 / 紫的联动是高风险组合，需要重点试玩。
