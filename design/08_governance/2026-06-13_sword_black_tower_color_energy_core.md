# 决策：确立《剑与黑塔》的彩能、色彩与武器卡池核心

日期：2026-06-13  
状态：已接受  
影响范围：产品 / 玩法 / 内容 / 叙事 / UI / 数据 / 技术实现

## 背景

在 [[inspiration/2026-06-13_background_story|对游戏背景设定的补充以及卡牌、战斗机制的重新设计]] 中，开发者提出了正式游戏名《剑与黑塔》、核后失色世界、黑塔、勇者联盟、左与茵，以及彩能、色彩附魔、左轮剑和机械臂等新机制设定。

本次确认后，该设定不再只是灵感补充，而成为当前设计基线。此前旧 MVP 中的“连锁层数、3 / 5 / 8 阈值、技能牌、卡牌包奖励”被新的彩能 / 色彩 / 武器卡池结构替代。

## 决策

采用《剑与黑塔》作为正式方向：

- 游戏名为《剑与黑塔》。
- 世界观采用核后失色世界、黑塔、勇者联盟、左与茵。
- 角色旅途与黑塔谜团提升为重要卖点，但不压过卡牌战斗。
- 剧情主要通过事件、卡牌、遗物、敌人和 Boss 文本隐晦表达，形成碎片化叙事。
- 彩能作为旧连锁层数的世界观命名和新版规则表达。
- 不再保留 3 / 5 / 8 固定阈值；终结牌按自身标注消耗彩能。
- 彩能可以通过行动牌附魔获得色彩。
- 卡牌类型改为行动牌和终结牌，不再保留技能牌。
- 起始结构改为武器选牌：玩家选择主手武器与副手武器，再从主手武器选 6 张、副手武器选 4 张，组成 10 张初始卡组。
- MVP 先支持左轮剑和机械臂。
- 普通战斗奖励改为色彩碎片 + 武器卡牌三选一，替代旧卡牌包。
- 五色机制成为基础战斗核心，并进入 MVP。
- 武器是角色机制和卡池入口，不是传统装备系统。
- MVP 敌人与 Boss 语境统一到黑塔世界观。
- MVP 加入简化 Boss 奖励。
- 当前 MVP 先打磨战斗系统，事件暂不进入 MVP；下一阶段优先做事件，高于商店和路线图。

## 理由

该方向把玩法与世界观绑定得更紧：玩家不是抽象地叠连锁，而是在失色世界中通过武器生成彩能、给行动牌附魔、用终结牌释放颜色与力量。彩能和色彩既能服务 combo 爽感，也能自然承接“夺回世界色彩”的叙事目标。

武器卡池让角色身份、战斗方式和构筑入口统一起来，避免传统装备刷装带来的范围膨胀。左轮剑和机械臂也能在 MVP 中形成清晰分工：输出 / 多段与防御 / 控制。

碎片化叙事让《剑与黑塔》的角色旅途成为重要卖点，同时保持卡牌战斗节奏。事件、卡牌和遗物文本承担剧情拼图，而不是用长剧情打断高频操作。

## 后续行动

- 重构数据 Schema，加入武器、色彩、附魔和彩能消耗字段。
- 重构规则层，支持彩能槽、五色附加效果、终结牌消耗彩能。
- 重构 UI，支持起始武器选牌、6 格彩能槽、色彩构成和附魔界面。
- 重构奖励流程，支持色彩碎片、武器卡牌三选一和 Boss 简化奖励。
- 迁移 MVP 敌人、卡牌、遗物和文本到黑塔世界观。
- 将旧实现中仍可复用的工程、数据加载、战斗日志和表现架构保留为技术基础。
