状态：持续维护

## 目的

统一项目内的术语、关键词和机制命名，避免文档、UI 和数据表使用不同表达。

| Run | 从开局到胜利或失败的一整局游戏流程。 | [[design/01_core_gameplay/01_run_structure]] |

| 左轮剑 | 起始武器之一，剑与枪结合，通过斩击生成彩能并以枪形态释放；可由玩家选择为主手或副手。 | [[design/02_content_systems/01_characters_and_archetypes]] |
| 机械臂 | 起始武器之一，高科技义肢，偏防御、护盾和彩能控制；可由玩家选择为主手或副手。 | [[design/02_content_systems/01_characters_and_archetypes]] |
| 魔物 | 本项目对战斗对手的正式通称，取代旧“恶魔”相关设定。系统实现和数据 ID 可继续使用 `enemy` 命名空间，但玩家可见文本和设计称呼优先使用“魔物”。 | [[design/02_content_systems/02_enemies_and_bosses]] |
| 行动点 | 旧自由出牌原型中的每回合战斗资源。三拍拆招 MVP 当前主要由拍位数量限制行动牌密度，行动点可作为兼容字段或后续扩展。 | [[design/01_core_gameplay/04_resource_economy]] |
| 拍位 | 三拍拆招中的编排单位。玩家拍位用于放入行动牌，魔物拍位用于展示本轮魔物动作。 | [[design/01_core_gameplay/02_combat_system]] |
| 玩家拍位 | 玩家本轮可编排行动牌的位置。第一版 MVP 中左轮剑和机械臂均为 3 拍；每拍最多放 1 张行动牌。 | [[design/01_core_gameplay/02_combat_system]] |
| 魔物拍位 | 魔物本轮动作序列中的一个动作块。玩家拍位可锁定未锁定魔物拍位进行动作对撞。 | [[design/01_core_gameplay/02_combat_system]] |
| 空门 | 某个魔物全部魔物拍位都已被锁定后，该魔物本体可被后续玩家拍位直接选择的状态。 | [[design/01_core_gameplay/02_combat_system]] |
| 动作 | 行动牌或魔物拍位内部的结算单元，包括斩击、钝击、弹射、格挡和闪避等。 | [[design/01_core_gameplay/03_card_system]] |
| 彩能 | 原“连锁层数”的世界观命名和新版规则表达；成功动作主要生成、终结牌主要消耗的轮内战斗资源，基础上限为 6，跨轮不保留。当前 MVP 只使用无色彩能。 | [[design/01_core_gameplay/02_combat_system]] |
| 无色彩能 | 不携带红、黄、蓝、绿、紫属性的彩能。第一版重构 MVP 的成功动作默认生成无色彩能。 | [[design/01_core_gameplay/02_combat_system]] |
| 色彩 | 后续阶段附着在彩能上的属性，由色彩碎片附魔行动牌后产生。色彩不是职业标签，而是卡牌用途的改写器。 | [[design/01_core_gameplay/03_card_system]] |
| 色彩碎片 | 后续普通战斗奖励之一，可为未附魔行动牌附魔；当前无色彩能切片暂不实现。 | [[design/02_content_systems/03_relics_items_rewards]] |
| 色彩核心 | 后续 Boss 或阶段性高价值奖励，用于提升某种色彩的附魔或终结结算效果。 | [[design/02_content_systems/03_relics_items_rewards]] |
| 带色彩能 | 后续五色阶段中携带红、黄、蓝、绿、紫之一的彩能，通常由已附魔行动牌生成。终结牌消耗带色彩能时触发对应颜色增强。 | [[design/01_core_gameplay/04_resource_economy]] |
| 色彩转译 | 颜色把卡牌基础效果改写成额外用途的规则，例如攻击转护甲 / 防御、攻击转治疗、防御转伤害或控制附带伤害。 | [[design/01_core_gameplay/03_card_system]] |
| 色彩改写 | 色彩转译的同义表达，用于描述“用什么颜色改写这张卡的用途”。正式规则文本优先使用“色彩转译”。 | [[design/01_core_gameplay/03_card_system]] |
| 行动牌 | 三拍拆招中的动作容器，放入玩家拍位后以内部动作串参与对撞。后续可被色彩碎片附魔。 | [[design/01_core_gameplay/03_card_system]] |
| 终结牌 | 放入终结槽，消耗彩能造成强大的伤害或效果的卡牌类型。五色阶段再根据被消耗彩能的色彩产生追加效果。 | [[design/01_core_gameplay/03_card_system]] |
| 防御 | 旧持续护甲资源或 UI 表达；当前三拍核心中优先表现为格挡动作，通过数值降低对应攻击动作伤害。 | [[design/01_core_gameplay/02_combat_system]] |
| 魔物意图 | 魔物在下一次行动中将执行的行为预告。 | [[design/01_core_gameplay/02_combat_system]] |
| 遗物 | 通常跨战斗生效、改变规则或提供被动收益的奖励，也承担碎片化叙事。 | [[design/02_content_systems/03_relics_items_rewards]] |
| 普通遗物 | 精英战斗在基础奖励之外额外给予的遗物类型，主要提供稳定构筑收益或小型规则变化。 | [[design/02_content_systems/03_relics_items_rewards]] |
| Boss 遗物 | Boss 战斗额外提供的高影响遗物类型，应能影响构筑、改变玩法或显著改变后续决策。 | [[design/02_content_systems/03_relics_items_rewards]] |
| 事件 | 非标准战斗节点中的选择型遭遇，是碎片化叙事的主要渠道之一。 | [[design/02_content_systems/04_events_and_encounters]] |
| 抽牌堆 | 玩家当前可抽取卡牌的牌区；抽牌堆不足时，默认将弃牌堆洗回抽牌堆继续抽。 | [[design/01_core_gameplay/02_combat_system]] |
| 弃牌堆 | 已使用、被弃掉或回合结束弃置的普通卡牌所在牌区；可在抽牌堆不足或特定效果下被洗回抽牌堆。 | [[design/01_core_gameplay/02_combat_system]] |

## 色彩术语

| 术语 | 定义 | 备注 |
| --- | --- | --- |
| 红色 | 进攻转化，提高伤害、追加伤害，或把防御 / 控制等非伤害效果的一部分转化为伤害。 | 不是攻击专属。 |
| 黄色 | 后续五色阶段的节奏转化，增加卡牌释放次数。黄色附魔效果不提供抽牌、返还行动点或回能量。 | 高风险，需防失控。 |
| 蓝色 | 防护转化，根据伤害或最终效果值获得护甲 / 防御，或提高防御效果。 | 不是防御专属。 |
| 绿色 | 生命转化，根据伤害或最终效果值回复生命。 | 回复不能超过最大生命。 |
| 紫色 | 效果翻倍，复制、翻倍、放大或延长异常效果。 | 高风险高收益，需防无限循环。 |

## 关键词候选

| 关键词 | 候选定义 | 备注 |
| --- | --- | --- |
| 斩击 | 攻击动作关键词，表示剑刃类伤害。 | 左轮剑基础动作。 |
| 钝击 | 攻击动作关键词，表示重击、拳击、砸击等冲击类伤害。 | 机械臂基础动作。 |
| 弹射 | 攻击动作关键词，表示弹丸、枪击或弹幕类伤害。 | 左轮剑终结牌使用。 |
| 格挡 | 防护动作关键词，按数值降低对应攻击动作伤害。 | 有效格挡可产无色彩能。 |
| 闪避 | 防护动作关键词，按概率或明确条件避开攻击动作。 | 当前 6 张 MVP 卡暂未使用，保留为动作体系关键词。 |
| 附魔 | 后续五色阶段将色彩碎片绑定到未附魔行动牌，使其后续生成对应色彩彩能。 | 需要区分同名牌实例。 |
| 最终伤害 | 一次结算在基础伤害、卡牌修正、易伤、牵引、后续颜色增强和其他加成计算后的伤害结果。 | 后续蓝色 / 绿色等可按最终伤害转译收益。 |
| 最终效果值 | 一次非伤害或混合效果在基础值、卡牌修正、后续颜色增强和其他加成计算后的统一数值，可作为护甲 / 防御、治疗或转伤害的计算基准。 | 用于后续多色终结牌结算。 |
| 牵引 | 负面状态候选：使本轮下一次单体攻击优先命中该目标，并提高该次伤害。 | 机械臂与左轮剑联动关键词。 |
| 裂痕 | 负面状态候选：使目标后续受到的伤害或多段弹幕收益被放大；具体层数和持续时间待调参。 | 紫色或左轮剑终结牌候选关键词。 |
| 易伤 | 负面状态：目标在持续期间受到的伤害提高。 | 重压等卡牌使用。 |
| 防御转伤害 | 将本次获得防御或当前防御的一定比例转化为伤害的效果。 | 红色防御牌、反击和护盾过载使用。 |
| 额外释放次数 | 让一张卡牌在本次打出时额外完整释放一次或多次的节奏收益。 | 黄色核心效果，需硬上限。 |
| 追加段数 | 让攻击、弹幕或释放效果增加额外段数的节奏收益。 | 与黄色额外释放次数区分；后续黄色不默认提供追加段数。 |
| 返还行动点 | 在旧自由出牌原型中获得行动点的效果。 | 不是后续黄色附魔效果；当前三拍切片不作为核心资源。 |
| 回费 | 在旧自由出牌原型中恢复或获得行动点的效果。 | 当前三拍切片不作为核心资源。 |
| 减费 | 降低旧行动点消耗或未来费用资源消耗的效果。 | 当前三拍切片不作为核心规则。 |
| 过牌 | 抽取、检索或以其他方式获得更多可用手牌的效果。 | 当前用于寻找关键行动牌和终结牌；后续也用于寻找附魔行动牌。 |
| 重洗 | 将弃牌堆洗回抽牌堆的牌区操作。 | 默认在抽牌堆不足时发生。 |
| 保留 | 使手牌、防御、彩能、终结槽或未来费用资源突破默认清空规则并延续到指定时点的效果。 | 保留彩能是高价值效果。 |

## 历史术语

| 术语 | 状态 | 说明 |
| --- | --- | --- |
| 连锁层数 | 已替代 | 被“彩能”替代。历史文档和旧实现中的连锁层数对应新版彩能的前身，但 3 / 5 / 8 阈值不再保留。 |
| 技能牌 | 已移除 | 旧卡牌类型。防御、控制、过牌、回费等职责现归入行动牌或终结牌。 |
| 卡牌包 | 已替代 | 旧普通战斗奖励。当前由武器卡牌三选一替代；后续五色阶段再接入色彩碎片奖励。 |

所有正式关键词必须补充定义、显示文本、图标需求和结算时机。
