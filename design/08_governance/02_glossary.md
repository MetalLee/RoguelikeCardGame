## 色彩术语

| 术语 | 定义 | 备注 |
| --- | --- | --- |
| 红色 | 进攻转化，提高伤害、追加伤害，或把防御 / 控制等非伤害效果的一部分转化为伤害。 | 不是攻击专属。 |
| 黄色 | 节奏转化，增加卡牌释放次数。MVP 黄色附魔效果不提供抽牌、返还行动点或回能量。 | 高风险，需防失控。 |
| 蓝色 | 防护转化，根据伤害或最终效果值获得护甲 / 防御，或提高防御效果。 | 不是防御专属。 |
| 绿色 | 生命转化，根据伤害或最终效果值回复生命。 | 回复不能超过最大生命。 |
| 紫色 | 效果翻倍，复制、翻倍、放大或延长异常效果。 | 高风险高收益，需防无限循环。 |

## 关键词候选

| 关键词 | 候选定义 | 备注 |
| --- | --- | --- |
| 附魔 | 将色彩碎片绑定到未附魔行动牌，使其后续生成对应色彩彩能。 | 需要区分同名牌实例。 |
| 最终伤害 | 一次结算在基础伤害、卡牌修正、易伤、牵引、颜色增强和其他加成计算后的伤害结果。 | 蓝色 / 绿色等可按最终伤害转译收益。 |
| 最终效果值 | 一次非伤害或混合效果在基础值、卡牌修正、颜色增强和其他加成计算后的统一数值，可作为护甲 / 防御、治疗或转伤害的计算基准。 | 用于多色终结牌结算。 |
| 牵引 | 负面状态候选：使本回合下一次单体攻击优先命中该目标，并提高该次伤害。 | 机械臂与左轮剑联动关键词。 |
| 裂痕 | 负面状态候选：使目标后续受到的伤害或多段弹幕收益被放大；具体层数和持续时间待调参。 | 紫色或左轮剑终结牌候选关键词。 |
| 易伤 | 负面状态：目标在持续期间受到的伤害提高。 | 重压等卡牌使用。 |
| 防御转伤害 | 将本次获得防御或当前防御的一定比例转化为伤害的效果。 | 红色防御牌、反击和护盾过载使用。 |
| 额外释放次数 | 让一张卡牌在本次打出时额外完整释放一次或多次的节奏收益。 | 黄色核心效果，需硬上限。 |
| 追加段数 | 让攻击、弹幕或释放效果增加额外段数的节奏收益。 | 与黄色额外释放次数区分；MVP 黄色不直接提供追加段数。 |
| 返还行动点 | 在当前回合获得行动点的效果。 | 不是 MVP 黄色附魔效果；可由具体卡牌、遗物或其他规则提供。 |
| 回费 | 在当前回合恢复或获得行动点的效果。 | 默认可使当前行动点超过基础上限。 |
| 减费 | 降低卡牌行动点消耗的效果。 | 影响范围需在文本明确。 |
| 过牌 | 抽取、检索或以其他方式获得更多可用手牌的效果。 | 用于寻找附魔行动牌和终结牌。 |
| 重洗 | 将弃牌堆洗回抽牌堆的牌区操作。 | 默认在抽牌堆不足时发生。 |
| 保留 | 使手牌、防御、行动点或彩能突破默认清空规则并延续到指定时点的效果。 | 保留彩能是高价值效果。 |

## 历史术语

| 术语 | 状态 | 说明 |
| --- | --- | --- |
| 连锁层数 | 已替代 | 被“彩能”替代。历史文档和旧实现中的连锁层数对应新版彩能的前身，但 3 / 5 / 8 阈值不再保留。 |
| 技能牌 | 已移除 | 旧卡牌类型。防御、控制、过牌、回费等职责现归入行动牌或终结牌。 |
| 卡牌包 | 已替代 | 旧普通战斗奖励。现由色彩碎片 + 武器卡牌三选一替代。 |

所有正式关键词必须补充定义、显示文本、图标需求和结算时机。
