# 魔物与 Boss

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义魔物和 Boss 的职责：它们不是障碍物，而是检验玩家武器构筑、三拍拆招、彩能生成、资源管理和战斗判断的题目。色彩附魔检验属于后续阶段。

## 魔物设计字段

- 名称。
- 世界观来源：黑塔异常、荒漠化区域、勇者联盟、失色生态或旧文明残留。
- 出现阶段。
- 生命与基础强度。
- 意图序列或行为池。
- 机制主题。
- 主要考察点。
- 克制与被克制关系。
- 奖励权重。
- 叙事碎片。

## Boss 设计字段

- 黑塔主题。
- 阶段主题。
- 规则变化。
- 压力升级方式。
- 对构筑的检验维度。
- 可读的预警反馈。
- 失败复盘点。
- Boss 奖励：Boss 遗物、Boss 特有卡牌或色彩核心。
- 与世界失色、茵、勇者联盟或黑塔真相的叙事关联。

## 第一版 MVP 遭遇节奏

第一版 MVP 使用 1 个线性章节，战斗顺序为：普通战斗 1 -> 普通战斗 2 -> 普通战斗 3 -> 精英战斗 -> 普通战斗 4 -> Boss 战斗。

前三场普通战斗负责教授基础三拍规则、魔物拍读取和武器卡牌奖励，并让玩家形成最小构筑变化。普通战斗 1 教拖行动牌入拍、自动瞄准魔物拍和成功产无色彩能；普通战斗 2 教格挡 / 闪避动作和机械臂防护职责；普通战斗 3 教多魔物、魔物拍锁定和空门本体目标。精英战斗负责检验攻防节奏、拍位目标分配和无色彩能生成稳定性，通过额外普通遗物给予规则收益。精英后的普通战斗负责让玩家试用遗物和新增卡牌，并在 Boss 前提供最后一次调整机会。Boss 战斗负责综合检验拍位规划、彩能积累、终结牌使用时机、防御和奖励构筑成果。

第一版 MVP 的 3 场战斗全部采用固定遭遇，不从魔物池随机抽取。固定遭遇用于降低测试变量，让团队先验证战斗循环、奖励闭环和数值压力是否成立。

## 第一版 MVP 魔物语境

旧版 MVP 魔物 ID 和美术资源可作为历史实现参考，但正式设计需要统一到《剑与黑塔》世界观。后续数据重构时，魔物可以保留稳定战斗职责，但名称、文本、立绘和叙事语义应迁移。

| 节点 | 战斗职责 | 新世界观方向 |
| --- | --- | --- |
| 普通战斗 1 | 教拖行动牌入拍、自动瞄准魔物拍和成功产无色彩能。低威胁单体。 | 黑塔外缘的失色残骸或勇者联盟训练用黑塔仿生靶。 |
| 普通战斗 2 | 教魔物拍位、格挡 / 闪避和防护职责。攻击 / 防御循环。 | 被黑塔影响的巡塔斥候，行动前会暴露明确意图。 |
| 普通战斗 3 | 教多魔物、目标锁定与空门本体。两个小型魔物。 | 黑塔周边分裂出的失色群体或荒漠化异形。 |
| 精英战斗 | 检验攻防节奏并给普通遗物。 | 腐败职业勇者、黑塔守卫或被联盟改造的塔下精英。 |
| 普通战斗 4 | 试用遗物和新卡，强化终结牌节奏。 | 色彩残留实验体或黑塔外围试炼者。 |
| Boss 战斗 | 综合检验，并给简化 Boss 奖励。 | 第一座黑塔的看守者，掌握失色和彩能异常的线索。 |

## 设计准则

- 普通魔物负责教授和变化，精英负责惩罚短板，Boss 负责综合考试。
- 魔物机制应尽量在行动前通过意图、状态或文案预告。
- Boss 不应硬性否定某个正常构筑，除非游戏提前提供应对路径。
- 魔物美术需要优先保证战斗信息可读：轮廓应清楚、朝向应稳定、体型差异应能帮助玩家快速区分普通魔物、精英和 Boss。
- 每个魔物都应至少携带一点世界观语义，服务黑塔、失色、勇者联盟或旧文明主题。
