# 角色与构筑原型

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义可玩角色、武器机制和构筑原型。角色是玩家理解卡池、世界观和策略身份的入口。

## 主角：左

「左」曾经是知名勇者「茵」的剑：左轮剑所诞生的剑灵。二人曾因理念分歧分道扬镳。后来在某次意外中，左作为剑灵面临消失危机；茵通过禁忌魔术牺牲自己的肉身，将自己的肉身与左的灵结合，使左成为人类。

成为人类的左感到内疚与后悔，决定独自实现茵生前的愿望：将世界恢复色彩。左带着茵遗留下的、也是自己曾经容器的左轮剑，前往一座座黑塔，寻找人类失去色彩的真相。左并不知道，茵的灵仍在左轮剑中沉睡。

## 角色设计字段

- 角色名称：左。
- 核心机制：使用主手和副手武器卡池生成彩能，并通过色彩附魔与终结牌兑现爆发。
- 初始生命：待数值阶段确认。
- 初始行动点：3。
- 初始彩能上限：6。
- 初始武器池：左轮剑、机械臂。
- 初始主副手：玩家在开局界面自由选择左轮剑或机械臂作为主手，另一个作为副手。
- 初始牌组：从主手武器 8 张初始牌中选 6 张，从副手武器 8 张初始牌中选 4 张。
- 初始遗物或能力：待定，可与茵、左轮剑或剑灵身份绑定。
- 主要构筑原型：输出彩能、护盾反击、控制终结、多段弹幕、色彩混合。
- 主要弱点：彩能跨回合不保留，终结牌需要稳定生成彩能和抽到正确牌。
- 学习难度：中等。

## 初始武器

### 左轮剑

剑与枪的结合。通过剑术斩击可以将彩能充入剑内，然后转变为枪形态，以彩能为子弹倾泻伤害。

构筑倾向：

- 高效生成彩能。
- 单体伤害与多段伤害。
- 终结牌爆发。
- 与红色、黄色、紫色有天然联动。

### 机械臂

高科技义肢，能生成粒子护盾，同时生产彩能，并可对敌人形成控制效果。

构筑倾向：

- 防御、护盾和反击。
- 控制、束缚和易伤。
- 稳定生成彩能。
- 与蓝色、绿色、紫色有天然联动。

## 构筑原型字段

- 原型名称。
- 关联武器。
- 启动条件。
- 核心收益。
- 主要风险。
- 推荐色彩。
- 关键卡牌或遗物。
- 对敌人机制的优势和劣势。

## 第一版 MVP 构筑原型候选

第一版 MVP 至少验证 3 种可辨认构筑方向。它们共享左轮剑和机械臂卡池，但对色彩碎片、行动牌附魔对象和终结牌选择有不同偏好。具体数值均为待调参，核心目标是验证 [[design/01_core_gameplay/03_card_system|卡牌系统]] 中“色彩改写用途，而不是绑定职业标签”的原则。

### 蓝绿重炮流

核心定位：用左轮剑打伤害，但不追纯红色，而是用蓝色和绿色把高伤害终结牌改造成护甲与治疗来源。

核心体验：

- 重炮不只是爆发牌，而是攻防治疗三合一。
- 蓝色重炮：造成伤害后，根据最终伤害获得护甲 / 防御。
- 绿色重炮：造成伤害后，根据最终伤害回复生命。
- 红蓝绿重炮：高伤害 + 高护甲 / 防御 + 治疗。

关键牌：

| 武器 | 关键牌 | 作用 |
| --- | --- | --- |
| 左轮剑 | 重炮 | 单体爆发核心 |
| 左轮剑 | 裂隙拔枪 | 击杀收割 |
| 左轮剑 | 装填斩 | 提高下一张终结牌伤害 |
| 左轮剑 | 枪剑校准 | 找终结牌，降低彩能压力 |
| 左轮剑 | 二连斩 | 快速生成彩能 |
| 左轮剑 | 快速突刺 | 0 费启动 |
| 机械臂 | 牵引抓取 | 锁定目标 |
| 机械臂 | 重压 | 易伤，提高最终收益 |

推荐附魔：

| 颜色 | 推荐对象 | 目的 |
| --- | --- | --- |
| 蓝色 | 二连斩、装填斩、快速突刺 | 生成蓝色彩能，让重炮转化护甲 / 防御 |
| 绿色 | 快速突刺、枪剑校准 | 生成绿色彩能，让重炮转化治疗 |
| 红色 | 装填斩、二连斩 | 少量提高伤害 |
| 紫色 | 重压、牵引抓取 | 放大终结前置收益 |

典型回合：

1. 牵引抓取锁定目标。
2. 重压施加易伤。
3. 蓝色二连斩生成蓝色彩能。
4. 绿色快速突刺生成绿色彩能。
5. 装填斩强化下一张终结牌。
6. 重炮消耗蓝 + 绿 + 其他彩能，造成高伤害，并获得护甲 / 防御与回复。

风险：

- 前期没有蓝绿彩能时只是普通重炮。
- 如果没有牵引抓取，单体爆发可能打错目标。
- 绿色回复需要生命上限限制。

### 红色机械防反流

核心定位：用机械臂防御牌生成红色彩能，把防御牌改造成攻击牌，再用反击 / 护盾过载收尾。

核心体验：

- 红色格挡：获得防御，同时造成基于本次防御值的伤害。
- 红色能量护幕：获得大量防御，同时造成大量红色转化伤害。
- 反击消耗红 / 蓝彩能后，既能打伤害，又能再获得护甲 / 防御。

关键牌：

| 武器 | 关键牌 | 作用 |
| --- | --- | --- |
| 机械臂 | 格挡 | 基础防御 + 红色伤害载体 |
| 机械臂 | 能量护幕 | 大防御 + 大红色转伤 |
| 机械臂 | 震荡掌 | 群体减攻 |
| 机械臂 | 反击 | 当前防御转单体伤害 |
| 机械臂 | 护盾过载 | 防御和彩能转群体伤害 |
| 机械臂 | 重压 | 易伤 |
| 左轮剑 | 快速突刺 | 0 费补彩能 |
| 左轮剑 | 装填斩 | 给终结补伤害 |

推荐附魔：

| 颜色 | 推荐对象 | 目的 |
| --- | --- | --- |
| 红色 | 格挡、能量护幕 | 防御牌变输出牌 |
| 蓝色 | 格挡、震荡掌 | 保持防御稳定性 |
| 黄色 | 快速突刺、震荡掌 | 增加卡牌释放次数 |
| 紫色 | 重压、束缚 | 放大异常或延长控制 |
| 绿色 | 维修协议 | 低血线容错 |

典型回合：

1. 红色格挡，获得防御并造成转化伤害。
2. 红色能量护幕，获得大量防御并造成转化伤害。
3. 震荡掌降低敌人攻击。
4. 重压施加易伤。
5. 反击消耗彩能，根据当前防御造成伤害。
6. 若反击消耗蓝色彩能，则根据最终伤害再次获得护甲 / 防御。

风险：

- 成型前伤害较慢。
- 依赖红色防御牌。
- 敌人若有穿透 / 持续伤害，需要绿色续航补救。
- 红色防御牌的转伤比例建议先以 50%-100% 作为待调参范围。

### 黄紫弹幕调色流

核心定位：利用黄色增加卡牌释放次数，以及紫色的放大 / 异常，让多段牌和 X 消耗终结牌反复触发色彩收益。

核心体验：

- 不是追求最高单次伤害，而是追求更多次出牌、更多段伤害、更多次色彩触发。
- 最适合围绕子弹风暴构筑。

关键牌：

| 武器 | 关键牌 | 作用 |
| --- | --- | --- |
| 左轮剑 | 子弹风暴 | X 彩能弹幕终结 |
| 左轮剑 | 二连斩 | 多段伤害 |
| 左轮剑 | 快速突刺 | 0 费启动 |
| 左轮剑 | 回旋剑舞 | 群体多段、快速攒彩能 |
| 左轮剑 | 枪剑校准 | 找终结牌 |
| 左轮剑 | 贯穿弹线 | 利用控制状态追加伤害 |
| 机械臂 | 束缚 | 控制敌人 |
| 机械臂 | 牵引抓取 | 锁定关键目标 |
| 机械臂 | 重压 | 易伤，放大多段伤害 |

推荐附魔：

| 颜色 | 推荐对象 | 目的 |
| --- | --- | --- |
| 黄色 | 快速突刺、二连斩、回旋剑舞 | 增加卡牌释放次数 |
| 紫色 | 束缚、重压、贯穿弹线 | 放大异常，制造爆发窗口 |
| 红色 | 二连斩、回旋剑舞 | 提高多段总伤害 |
| 蓝色 | 任意高频产彩能牌 | 让弹幕终结后获得护甲 / 防御 |
| 绿色 | 快速突刺或二连斩 | 让弹幕终结附带回血 |

典型回合：

1. 黄色快速突刺，0 费启动，生成黄色彩能。
2. 黄色二连斩，造成两段伤害，生成黄色彩能。
3. 紫色重压，施加强化易伤，生成紫色彩能。
4. 牵引抓取锁定关键目标。
5. 枪剑校准寻找子弹风暴。
6. 子弹风暴消耗黄 + 黄 + 紫 + 红 + 蓝等彩能。
7. 黄色增加子弹风暴或前置多段牌的释放次数；紫色施加裂痕或放大后续弹幕；蓝色根据最终总伤害获得护甲 / 防御。

风险：

- 动画可能过长。
- 触发太多会让玩家看不懂。
- MVP 建议限制黄色每张牌最多增加 1 次额外释放；子弹风暴最多因黄色增加 1 次额外整牌释放，具体次数待试玩调参。
- 紫色复制 / 放大不能造成无限循环。

## 设计准则

- 每个武器至少应支持 2 到 3 个可辨认构筑方向。
- 原型之间可以共享组件，但不应完全抢同一组最优牌。
- 角色差异应优先来自规则、武器和资源结构，而不是只换卡面。
- 角色叙事应通过卡牌、遗物、事件和 Boss 文本逐步显露，不应在开局用长篇说明阻断玩家。
