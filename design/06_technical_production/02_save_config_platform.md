# 存档、配置与平台

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义存档、设置、平台目标和用户配置要求。

## 存档需求

- 当前 Run 进度。
- 解锁状态。
- 设置项。
- 统计数据。
- 内容版本。

## 设置需求

- 音量分类。
- 文本语言，待定。
- 动画速度或减少动画。
- 分辨率和窗口模式。
- 输入配置，待定。

## 平台目标

首版平台待定。独立卡牌 Roguelike 通常优先 PC，后续再评估掌机、移动或主机适配。

## 设计准则

- 存档失败不能导致玩家长期进度丢失。
- 设置项应服务高频体验痛点，避免首版过度扩张。
- 平台适配不应早于核心循环验证。
