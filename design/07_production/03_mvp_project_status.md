## 游戏概念摘要

《剑与黑塔》是一款以武器卡池构筑、三拍拆招、彩能积累和终结牌爆发为核心的 Roguelike 卡牌独立游戏。玩家扮演由剑灵化为人类的「左」，带着沉睡着「茵」之灵的左轮剑，探索核后失色世界中的黑塔，逐步拼合世界失去色彩的真相。色彩附魔是后续构筑扩展方向。

核心体验链路：

1. 玩家选择左轮剑 / 机械臂的主副手顺序，系统按固定规则自动组成初始卡牌。
2. 玩家在战斗中把行动牌放入拍位，读取并拆解魔物拍位。
3. 成功造成伤害、有效格挡或成功闪避时积累无色彩能。
4. 玩家使用终结牌消耗彩能并触发终结演出；后续阶段再叠加色彩效果。
5. 玩家通过卡牌、遗物、事件和魔物文本拼合黑塔谜团。

## 第一版 MVP 新范围

第一版 MVP 目标是验证新战斗局内循环，而不是验证完整 Roguelike 内容量。

新 MVP 范围：

- 目标时长：约 5-8 分钟。
- 流程结构：1 个线性章节。
- 战斗结构：1 场单目标普通战斗、1 场多目标普通战斗、1 场 Boss / 精英强度战斗。
- 遭遇策略：3 场战斗全部采用固定遭遇，并统一到黑塔世界观。
- 角色范围：左。
- 武器范围：左轮剑 + 机械臂；玩家自由选择其中一个作为主手、另一个作为副手。
- 初始牌组：每把武器固定 4 张初始牌，主手与副手全部带入，组成 8 张。
- 卡牌类型：行动牌、终结牌。
- 彩能规则：基础上限 6，当前只使用无色彩能，跨轮不保留。
- 色彩规则：红、黄、蓝、绿、紫 5 种色彩和行动牌色彩碎片附魔延后到五色阶段。
- 奖励规则：当前普通战斗只验证武器卡牌三选一；精英额外普通遗物、Boss 战后奖励和色彩碎片附魔奖励均延后。
- 生命规则：MVP 中每场战斗开始时生命回满，优先验证单场战斗和奖励闭环。
- 胜负规则：生命归零结束本次 MVP Run；击败 Boss 显示 MVP 通关并提供重开入口。

## 已替代的旧规则

以下旧 MVP 规则已被 2026-06-13 新设计替代：

- “连锁层数”作为 UI / 规则术语。
- 3 / 5 / 8 固定阈值。
- 行动牌 / 技能牌 / 终结牌三类卡牌。
- 技能牌默认不消耗行动点、不加连锁的分类。
- 普通战斗后的行动牌包 / 技能牌包 / 终结牌包奖励。
- 前两次普通战斗终结牌包保底群体终结牌。

## 下一步建议

1. 继续打磨三拍拆招 UI：魔物拍显示、自动瞄准、取消回手牌、结算反馈和终结槽释放提示。
2. 完善规则层：拍位长度、动作对撞边界、成功产无色彩能、终结牌重复释放和轮末清理。
3. 收敛奖励流程：当前优先武器卡牌三选一；精英普通遗物、Boss 简化奖励和色彩碎片延后。
4. 重构 UI：6 格无色彩能槽、玩家拍位、魔物拍位、终结槽、动作串 tooltip 和起始主副武器选择界面。
5. 将 3 场魔物和文本统一到黑塔世界观，并让每场战斗承担明确三拍教学职责。
6. 持续运行数据校验、规则层测试和 Godot 启动验证。
