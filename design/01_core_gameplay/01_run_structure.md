# 单局结构

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义一局游戏的阶段结构、节点类型、节奏曲线和失败条件。

## 标准 Run 结构

- 起始选择：角色、初始牌组、初始奖励或路线预览。
- 早期：教学式压力，鼓励建立初步构筑方向。
- 中期：敌人机制开始针对构筑短板，奖励提供转向或强化。
- 后期：资源压力增加，路线选择更高风险。
- 终局：Boss 或特殊挑战检验构筑完整度与操作理解。

## 时长目标

完整版本的单局目标时长为 30-40 分钟。Run 结构应服务中等时长逐步成型：早期让玩家识别方向，中期形成可运行联动，后期提供高层数连锁和终结卡爆发窗口，终局检验构筑完整度。

节点数量、战斗时长、奖励频率和路线分支应围绕该时长校准。单局过短会削弱构筑成型和 combo 爽感，单局过长会降低反复开局意愿。

## 第一版 MVP Run 范围

第一版 MVP 采用 1 个线性章节，目标时长约 10-15 分钟，主要验证战斗局内循环。战斗顺序为：普通战斗 1 -> 普通战斗 2 -> 普通战斗 3 -> 精英战斗 -> 普通战斗 4 -> Boss 战斗。6 场战斗全部采用固定遭遇，等战斗循环成立后再加入普通战斗随机池。

第一版 MVP 暂不做路线图、事件、商店、休息或完整货币经济。地图和路线选择在后续垂直切片阶段补充；MVP 阶段先保证行动点、连锁层数、终结牌、抽弃牌循环、防御结算、卡牌包奖励和精英普通遗物能够形成可玩的闭环。后续完整 Run 的阶段性 Boss 再验证特殊遗物三选一。

第一版 MVP 内容策略采用极简小型教学池：初始牌组 10 张，奖励牌约 9 张，其中行动牌、技能牌和终结牌各 3 张；普通遗物 1 个；6 场固定遭遇只做必要的 6 组敌人。该内容池只服务教学递进、奖励选择和最小构筑成长，不追求完整流派、完整构筑原型或长期重复游玩变化。具体 9 张奖励牌、1 个普通遗物和 6 组敌人内容进入 MVP 开发过程中再确认。

第一版 MVP 每场战斗开始时玩家生命回满，优先验证单场战斗、固定遭遇和奖励闭环。正式版本方向为生命跨战斗保留且不自动恢复；本轮不展开正式版本恢复节点或恢复经济。

第一版 MVP 中 Boss 击败即显示 MVP 通关并提供重开入口，不做 Boss 战后奖励、局外奖励或解锁。后续完整 Run 的阶段性 Boss 可继续承担构筑转折点奖励职责。

前 3 场普通战斗用于建立基础规则理解并提供 3 次卡牌包奖励；精英战斗用于暴露构筑短板并给予普通遗物；精英后的普通战斗用于让玩家感受遗物和新增卡牌的影响，并在 Boss 前获得最后一次卡牌包调整机会；Boss 战斗用于综合检验构筑和战斗判断。

第一版 MVP 的六场固定战斗采用教学递进型职责分配：普通战斗 1 教行动点 / 出牌 / 连锁，普通战斗 2 教敌人意图与防御，普通战斗 3 教多敌人与目标选择，精英战斗检验攻防节奏，普通战斗 4 让玩家试用遗物和新卡，Boss 战斗综合检验。普通战斗 1 后和普通战斗 2 后的终结牌包都固定保底 1 张群体攻击终结牌，以支撑普通战斗 3 的多敌人教学；外层仍不明牌，玩家仍需主动选择终结牌包。

## 节点类型

- 普通战斗：提供基础卡牌包奖励和构筑调整机会；玩家从行动牌包、技能牌包、终结牌包中选择 1 包打开，选择前只知道卡牌包类型，不知道具体卡牌，并从打开后的 3 张同类型候选卡牌中选择 0-3 张加入卡组。第一版 MVP 采用固定可重复池，每类卡牌包固定显示该类型 3 张奖励牌，拿过的牌仍可再次出现并重复加入同名牌。
- 精英战斗：高风险高回报，检验当前强度；除卡牌包外，额外给 1 个普通遗物。第一版 MVP 只有 1 个普通遗物，因此该奖励等同固定获得；后续完整版本再扩展为普通遗物随机池。
- Boss：阶段性规则考试；后续完整 Run 中可奖励稀有度更高的卡牌包，并额外提供 3 个能够影响构筑、改变玩法的特殊遗物候选，玩家选择 1 个获得。第一版 MVP 的终点 Boss 击败后直接胜利，不发放 Boss 战后奖励。
- 事件：提供信息不完全的风险回报选择。
- 商店：资源转换和定向补强。
- 休息或锻造：恢复与强化之间的取舍。
- 宝箱或特殊奖励：节奏缓冲与惊喜。

## 失败条件

- 生命归零。
- 第一版 MVP 中，生命归零立即结束本次 MVP Run，回到开局或失败结算；玩家流程不提供当前战斗重试。
- 特殊模式或挑战条件失败，待定。

## 设计准则

- 每个阶段都应有不同的压力主题。
- 普通战斗奖励应帮助玩家定向补强构筑，同时通过类型选择保留方向判断、通过开包保留随机性，并允许选择 0 张以控制卡组厚度。
- 普通战斗卡牌包至少包含 1 张高于普通稀有度的卡牌；早期以普通牌为主，中后期逐步提高高稀有卡牌出现概率。
- 精英和 Boss 不只是更高数值；第一版 MVP 中精英通过普通遗物提供机制差异，后续完整 Run 中 Boss 再通过特殊遗物和更高质量卡牌包提供阶段性构筑转折。
- Boss 特殊遗物应能影响构筑或改变玩法，并通过三选一成为后续完整 Run 的阶段性主动构筑转折点。
- 完整版本的路线图应让玩家能做战略规划，但保留局部未知；第一版 MVP 暂不做路线图。
- 完整版本的构筑成型和高层数连锁高光应在 30-40 分钟目标时长内自然出现；第一版 MVP 只验证最小构筑成长和终结牌兑现是否成立。
