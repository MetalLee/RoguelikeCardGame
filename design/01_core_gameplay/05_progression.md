# 成长与解锁

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义局内成长、局外解锁和玩家长期目标。成长系统应增强重玩动机，但不能掩盖核心战斗和构筑质量。

## 局内成长

- 卡牌获得、升级、删除和转化。
- 普通战斗后的基础卡牌获得采用卡牌包结构：从行动牌包、技能牌包、终结牌包中选择 1 包打开；选择前只知道卡牌包类型，不知道具体卡牌；打开后再从 3 张同类型候选卡牌中选择 0-3 张加入卡组。
- 第一版 MVP 的 9 张奖励牌采用固定可重复池；每次打开某类型卡牌包都显示该类型固定 3 张奖励牌，已经拿过的牌仍可再次出现，并允许重复加入同名牌。
- 每个卡牌包至少包含 1 张高于普通稀有度的卡牌；早期以普通牌为主，中后期逐步提高高稀有卡牌出现概率，形成局内奖励质量成长。
- 少拿或不拿卡牌默认没有额外补偿，作为控制卡组厚度的纯构筑取舍；跳过收益留到商店、货币和删牌经济确定后再评估。
- 遗物或被动效果积累；精英战斗额外给 1 个普通遗物。第一版 MVP 只有 1 个普通遗物，因此该奖励等同固定获得；后续完整版本再扩展为普通遗物随机池。后续完整 Run 的阶段性 Boss 战斗额外提供 3 个能够影响构筑、改变玩法的特殊遗物候选，玩家选择 1 个获得。
- 后续完整 Run 的阶段性 Boss 战斗奖励稀有度更高的卡牌包，用于提供阶段性高质量卡组补强；第一版 MVP 的终点 Boss 击败即通关，不发放 Boss 战后奖励。
- 最大生命、资源上限或特殊能力变化。
- 构筑原型逐步成型。

## 局外成长

待定。可选方向包括角色解锁、卡牌池解锁、挑战模式、难度层级、图鉴和成就。

第一版 MVP 不做局外奖励、局外解锁、Meta 成长、通关奖励或新内容解锁。

## 设计准则

- 局外解锁应扩大选择空间，而不是单纯提高基础数值。
- 新内容解锁不应破坏新玩家学习节奏。
- 局内卡牌获得应允许玩家控制卡组厚度，避免奖励系统强迫玩家拿不适合当前构筑的牌。
- 默认不把少拿或不拿卡牌转化为额外成长资源，避免局内成长路径过早偏向经济套利。
- Boss 特殊遗物应通过三选一成为构筑方向变化或玩法变化的关键成长点，而不是单纯提高数值。
- 长期目标应鼓励尝试不同构筑和路线。
- 失败奖励需要谨慎，避免玩家觉得必须刷失败才能变强。

## 待验证问题

- 是否需要 Meta 养成。
- 解锁内容是否影响平衡池。
- 难度层级如何递增。
