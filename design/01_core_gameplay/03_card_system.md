# 卡牌系统

状态：草案  
上级索引：[[design/README|Design Knowledge Base]]

## 目的

定义卡牌类型、稀有度、关键词、升级、获得和删除规则。

## 卡牌基础字段

- 名称。
- 类型：攻击 / 防御 / 技能 / 能力 / 特殊，待定。
- 费用。
- 效果文本。
- 连锁条件：若卡牌在指定连锁层数下获得额外效果，需要明确阈值和额外结算。
- 目标规则。
- 稀有度。
- 标签。
- 升级版本。
- 解锁条件，若有。

## 卡牌设计准则

- 卡牌文本应先说明主要效果，再说明条件或副作用。
- 单卡强度不能只靠数值判断，还要看构筑依赖、时机限制和机会成本。
- 每个构筑原型都需要启动牌、终结卡、防御方案和资源方案。
- 终结卡应能在高连锁层数下兑现显著收益，例如巨额伤害、范围打击、追加抽牌、资源返还、复制或特殊动画演出。
- 终结卡默认在结算后使连锁层数归零；特殊终结卡可以保留部分或全部连锁层数，但必须承担稀有度、费用、阈值或副作用等成本。
- 终结卡数量由玩家自由决定，但过多终结卡会挤压启动牌、延展牌和防御牌比例，提高无法获得连锁层数或无法兑现爆发的卡手风险。
- 默认卡组和前期推荐围绕 1-2 张终结卡构筑；中后期若过牌、减费、回费、复制或连锁保留能力提升，可以允许玩家加入更多终结卡。
- 终结卡之间可以存在联动关系，支持多终结卡流派，但这类流派必须依赖足够的卡组基础设施，而不是无条件强于单终结卡构筑。
- 启动牌和过渡牌应帮助玩家提升连锁层数、调整牌序或寻找终结卡，而不是只提供孤立数值。
- 低稀有度卡牌负责稳定基础策略，高稀有度卡牌负责改变打法或放大组合。
- 删除、升级、复制和转化卡牌属于高影响系统，需谨慎控制频率。

## 终结卡效果空间

终结卡在高连锁条件下可以使用多种额外效果。首要目标是让玩家感到“高连锁被兑现”，而不是把所有终结卡都做成同一种伤害倍率。

- 输出类：伤害增加、暴击率增加、暴击伤害提高、群体伤害、多段伤害。
- 连锁类：保留部分或全部连锁层数，或在结算后返还一定连锁层数。
- 资源类：抽牌、回费、减费、复制、检索或临时生成卡牌。
- 控制类：赋予敌人异常状态、削弱敌人、延迟敌人行动或破坏敌人防御。
- 增益类：赋予自身增益、防御、护盾、下回合资源或持续强化。
- 特殊类：秒杀非 Boss 敌人、改变目标规则、触发额外结算或与另一张终结卡联动。

数值表达可以采用固定值、基于连锁层数的加算，或基于连锁层数的乘算。乘算和秒杀类效果属于高风险设计，需要更严格的阈值、目标限制和测试。

基础终结卡效果候选包括单体高伤害、群体伤害、暴击率提升、抽牌 / 回费。具体首版卡池和效果清单待其他设计维度确认后再定。

## 连锁阈值

当前连锁阈值基准采用 3 / 5 / 8 三档，分别对应学习、续航和成型爆发。

- 3 层低阈值：用于教学和早期构筑，让玩家用较低成本理解连锁机制。适合单体高伤害、基础暴击率提升等简单效果。
- 5 层中阈值：用于熟练后的续航连锁。适合抽牌、回费、减费或帮助继续维持连锁的终结卡。
- 8 层高阈值：用于卡组成型后的高光爆发。适合巨额单体伤害、巨额群体伤害，或后续扩展的秒杀非 Boss 敌人等高冲击效果。

同一张终结卡可以只使用一个阈值，也可以在不同阈值下逐级增强，但早期验证应优先保证文本和反馈清晰。

连锁层数不设置硬上限。3 / 5 / 8 是关键奖励阈值，不是层数上限；超过 8 层后的收益曲线、显示方式和数值约束后续定义。

## 连锁相关卡牌定位

- 启动牌：低成本提高连锁层数，帮助一回合开始运转。
- 延展牌：提供抽牌、减费、回费、复制或临时生成，让玩家继续维持连锁。
- 终结卡：在高连锁阈值下触发额外强力效果，是 combo 的主要兑现点，默认结算后清空连锁层数。
- 稳定牌：帮助玩家在未抽到终结卡时仍能防御、过牌或维持基础输出。
- 反压牌：在敌人高压回合中允许玩家放弃极限连锁，转向防守或恢复节奏。

## 终结卡密度

终结卡密度是玩家构筑中的主动取舍，而不是硬性上限。

- 前期基准：默认卡组或前期构筑推荐 1-2 张终结卡，保证玩家能稳定启动、叠连锁和防御。
- 中期扩展：当卡组获得稳定过牌、减费、回费、复制或检索能力后，可以承载更多终结卡。
- 后期流派：高过牌或高连锁保留构筑可以围绕多张终结卡形成连续爆发，终结卡之间也可以互相补充或联动。
- 风险表达：奖励选择、牌库查看和战斗反馈应让玩家意识到终结卡过多会增加卡手风险。

## 待验证问题

- 初始牌组大小。
- 每局卡牌获得频率。
- 卡牌池总量和首版最小数量，待其他设计维度确认后再定。
- 升级是否只改数值，还是允许改机制。
- 连锁阈值 3 / 5 / 8 层是否足够清晰。
- 高连锁额外效果应写在卡牌文本中，还是通过关键词和悬浮信息解释。
- 特殊终结卡保留连锁层数时，应保留固定数量、百分比，还是全部保留。
- 不同阶段的推荐终结卡密度是否需要通过 UI 或新手引导提示。
- 单体高伤害、群体伤害、暴击率提升、抽牌 / 回费等候选终结卡在 3 / 5 / 8 阈值下应采用哪些数值。
- 连锁加算、连锁乘算和固定值效果的数值边界。
- 超过 8 层后是否继续提供线性、递减或仅展示性的收益。
